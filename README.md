# Bank-Management-System

### It's a project based on CLI (Command Line Interface)

It allows Users to -

    1. Creat an ACCOUNT.
</br>

    2. LOGIN to the System, then-
           2.1) Add Amount.
           2.2) Check Balance.
           2.3) Withdraw Cash.
           2.4) Tranfer Amount.
           2.5) Edit Profile.
           2.6) Logout from the system.
